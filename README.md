# Configurable-Confirm-Box
Provides option to customize the confirmation box.

Options:

-Provides options to customize message, headerstyle, button style ,containerstyle.

Like:
                 message: {
											'font-size':'18px',
											'font-weight':'bold',
											'color':'green',											},
								  headerStyle:{
											'text':'Custome Confirm Alert.....',
											'font-size':'18px',
											},
								  btnStyle: {
											'font-weight':'bold',
											'color':'red',
											},
								  containerStyle{
								  }
